# springboot数据翻译演示demo

## Quick Start

http://localhost:8080/index.html

## 功能特性

* 简单springmvc特性
* 能把生成的代码直接运行起来
* 没有权限控制
* 具有完整的：简单的增、删、改、查

## 使用

* 想要easyui就copy生成代码中的easyui部份
* 想要jqgrid就copy生成代码中的jqgrid部份
