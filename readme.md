# springboot数据翻译组件
即支持翻译数据属性，并把翻译结果存到数据库，加载时自动缓存翻译结果1分钟，以减少频繁调用数据库。

## Quick Start
http://localhost:8080/index.html  
http://localhost:8080/doc.html

## 功能特性

* 支持springboot,springmvc
* 支持mysql,postgresql等
* 支持easyui,jqgrid,layui前端对功能翻译的修改
* 支持java注解使用
* 支持百度,有道，google等翻译api
* 支持查询时用guava缓存翻译结果
